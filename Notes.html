Class 1

-> Variables: They are like boxes or containers which store data/information Rules on naming the variables:
  • ->Variables can't start with number or symbol
  • ->They can only contain '' (underscore) , alphabets and numbers
  • ->They can start with '' (underscore) and alphabets
-> DataTypes: They tell what kind of data it is (like if something is a number or a string)
  • -> Characters (char is shortform): Any kind of letter (includes alphabets, numbers, symbols, emojis)
  • -> Strings (str is shortform): A row or a string of characters (like a necklace with characters as beads)
  • -> Integers (int is shortform): Any number (without decimal point) is an integer. (you can do maths using integers)
  • -> Float (has no shortform): Any number with a decimal point.
  • -> Boolean (bool is shortform): Yes or No (in python its True or False)
-> You can store all data types in variables and also make your own data types

print() : It is a function in python which gives output. Anything written inside the () is outputed
input() : It is a function in python which asks for an input from the user.

if:(statement) - This statement is used for comparision.
else:(statement) - Also used for comparision. But after the if: statement

Example

          x=4
          y=3

          if x==y:
            print("they both are equal")
          else:
            print("they both are not equal")
